# Pocket Agent v0.2.1

**Release Date**: 2026-05-06

OpenClaw backend support — connect to OpenClaw Gateway for multi-agent routing with per-agent session isolation.

## New Features

### OpenClaw Backend Support
- **Dual-backend architecture** — toggle between Hermes and OpenClaw by changing `.env` variables. No code changes needed.
- **Multi-agent routing** — `API_AGENT=zhenyan` routes to OpenClaw agent `zhenyan` via `model: "openclaw/zhenyan"`. Agent name can be set to any configured OpenClaw agent ID.
- **Per-agent session isolation** — session keys follow the format `agent:{agent}:pocket-agent-YYYY-MM-DD`, keeping each agent's conversation history separate and identifiable in `openclaw status`.

### Environment-Driven Configuration
- **API URL moved to ENV** — `api_url` removed from settings UI and `AppConfig`. PA reads `API_SERVER` from `.env` file at startup. This simplifies deployment and removes the risk of users accidentally misconfiguring the endpoint.

## Improvements

### Session ID Format
- **Agent-aware session keys** — OpenClaw mode sends `x-openclaw-session-key: agent:{agent}:pocket-agent-YYYY-MM-DD`, readable at a glance in OpenClaw session lists.
- **Hermes unchanged** — still uses `X-Hermes-Session-Id: pocket-agent-YYYY-MM-DD`. Full backward compatibility.
- **Daily rotation preserved** — date-based suffix continues to auto-rotate sessions at midnight.

### Error Message Generalization
- Hardcoded `"连接 Hermes 失败"` and `"Hermes 返回错误"` replaced with generic `"接口连接失败"` and `"接口返回错误"`, since the same code now serves both backends.

## Bug Fixes

- **Session header overriding model routing** — `x-openclaw-session-key` header was found to override OpenClaw's `model` field routing. Fixed by prepending `agent:{agent}:` to the session key, keeping model-based routing authoritative.
- **Missing `stream: true` in body** — briefly removed during debugging, restored as SSE streaming is required by PA's eventsource parser.

## Technical Details

### Routing Flow

```
.env: API_AGENT=zhenyan
  ↓
send_message() → get_api_agent() = Some("zhenyan")
  ↓
chat_stream():
  model = "openclaw/zhenyan"              ← routes to zhenyan agent
  x-openclaw-session-key: agent:zhenyan:...  ← session isolation
  ↓
OpenClaw Gateway routes to agent:zhenyan
```

### Hermes Fallback (No ENV Change)

```
.env: API_AGENT unset
  ↓
chat_stream():
  model = "default"
  X-Hermes-Session-Id: pocket-agent-YYYY-MM-DD
  ↓
Hermes api_server — unchanged behavior
```

### .env Configuration

```ini

# API_AGENT=zhenyan
```
