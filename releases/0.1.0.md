# Pocket Agent v0.1.0

**Release Date**: 2026-05-03

First public release. A minimal desktop AI voice companion built with Tauri 2.0 + Svelte 5 + Rust.

## Core Features

### Voice Interaction
- **Push-to-talk via fn key** — toggle mode: press to start, press again to stop
- **Whisper STT** — local speech-to-text with auto language detection (Chinese, English, Japanese, Korean)
- **Edge TTS playback** — free, unlimited text-to-speech via Microsoft Edge TTS
- **Multi-language voices** — configurable primary + 2 auxiliary voices with automatic language matching
- **30s max recording** with auto-cutoff, 1.5s minimum to prevent Whisper hallucination
- **Escape to cancel** — discard recording without processing
- **Audio prewarm** — initializes capture device on startup to avoid missing first words

### AI Chat
- **SSE streaming** — real-time response from Hermes Agent gateway
- **Automatic language detection** — Unicode character analysis selects the right TTS voice
- **Emotion-aware display** — keyword-based emotion detection (friendly, cheerful, calm, serious, sad, excited, angry) drives typewriter speed
- **Retry logic** — up to 2 retries on SSE connection failure, 5 consecutive error threshold
- **Local command execution** — opt-in `[CMD:...]` tags in LLM responses (disabled by default, requires `ENABLE_LOCAL_COMMANDS=true`)

### UI
- **Floating avatar** — drag to move, click expand button to toggle chat panel
- **Dark sci-fi aesthetic** — compact 220x360 window, glass-effect panels
- **Dynamic Island** — recording/thinking state indicator below avatar
- **Settings panel** — tabbed layout (General / Voice) with grouped voice dropdowns
- **System tray** — macOS menu bar with Settings and Quit
- **Window position persistence** — remembers screen position across restarts
- **Accessibility detection** — auto-checks CGEventTap permission, guides user to System Settings

### Technical
- **Tauri 2.0** — native macOS app, no Electron
- **Svelte 5 runes** — `$state`, `$derived`, `$effect` (no legacy reactive statements)
- **Rust backend** — cpal audio capture, rodio playback, reqwest SSE client
- **macOS CGEventTap FFI** — raw keyboard event listener for global hotkey (no rdev dependency)
- **Environment config** — `.env` file for API key, session ID, TTS binary path

## Requirements
- macOS 12+ (Monterey)
- Hermes Agent gateway running on localhost:8642
- Edge TTS binary (`pip install edge-tts`)
